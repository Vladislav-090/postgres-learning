# PostgreSQL Learning

This repository contains my practical PostgreSQL learning tasks and exercises.

The goal of this repository is to build a strong foundation in SQL and PostgreSQL before moving to backend projects with Go and PostgreSQL.

## Topics Covered

* Basic SELECT queries
* INSERT, UPDATE, DELETE
* Table constraints
* ALTER TABLE
* Primary keys
* Foreign keys
* JOIN operations
* Aggregation
* Transactions

## Repository Structure

* `001-users-basics` — basic table creation and SELECT queries
* `002-update-and-delete` — updating and deleting records
* `003-constraints` — NOT NULL, UNIQUE, CHECK, DEFAULT
* `004-alter-table` — changing table structure
* `005-foreign-key` — relations between tables
* `006-join` — JOIN queries
* `007-aggregation` — COUNT, SUM, AVG, MIN, MAX, GROUP BY
* `008-transactions` — BEGIN, COMMIT, ROLLBACK

Each task contains SQL practice focused on one specific topic.

## Purpose

This repository is part of my backend development learning path.

I use it to practice SQL manually and understand how data is stored, changed, connected, and queried in relational databases.

The knowledge from this repository is later applied in Go backend projects such as REST APIs with PostgreSQL.
